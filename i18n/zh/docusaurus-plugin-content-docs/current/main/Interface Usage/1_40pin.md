# 1. 40PIN引脚

ArmSoM 产品的 40Pin 接口设计，虽然常用接口（如 I2C、SPI、UART、PWM）的位置保持一致，但由于不同芯片的资源分配，部分引脚可能会空置或未定义功能。
开发者在使用时，可以根据具体产品的40PIN的定义来使用。

:::tip
ArmSoM 40pin 的5v不支持输入。
:::
