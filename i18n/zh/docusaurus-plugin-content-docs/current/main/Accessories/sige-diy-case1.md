# Sige Diy Case 1

ArmSoM-Sige 外壳将高品质结构与简洁、简约的设计相结合，金属外壳使其更高端。

它们既能保护您的 Sige 产品，同时有一定的散热功能，有紫色或黑两种颜色可供选择。

![ArmSoM-sige7-Metal-shell](/img/sige/sige7-Metal-shell.jpeg)

## 技术规格

- 外壳材质:铝材(AL6063)
- 表面处理:喷砂氧化
- 外壳颜色:紫色 / 黑色
- 重量:0.111kg
- 外壳尺寸:101.5*66*23mm

## 兼容性

- Sige1 3mm散热垫
- Sige3 3mm散热垫
- Sige5 3mm散热垫
- Sige7 2mm散热垫
