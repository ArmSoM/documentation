# ArmSoM Camera module 1

ArmSoM 使用了高性能的超清晰4K相机 OV13850。

OV13850 模块具有以下特点：
* 像素尺寸：1.12 毫米 x 1.12 毫米
* 1300 万像素，30 帧每秒
* 支持图像尺寸为 13.2MP（4224 x 3136）、10MP（4224 x 2376）、2K（2112 x 1568）、EIS1080P（2112 x 1188）、EIS 720P（1408 x 792）等
* 双线串行总线控制（SCCB）
* 高达 4 通道 MIPI 串行输出
* 工作温度范围：0 ℃～60 ℃，存储温度范围：-30 ℃～80 ℃
